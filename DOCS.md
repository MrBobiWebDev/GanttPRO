## 📋 Краткая информация

### Системные требования
- Java 17+
- Maven 3.6+
- Браузер (любой современный)

### Учетные данные демо-пользователя
- Email: `demo@example.com`
- Пароль: `demo123`

### Основные URL
- Приложение: http://localhost:8080
- API: http://localhost:8080/api/
- H2 Console: http://localhost:8080/h2-console

### Команды для быстрого запуска

**Windows:**
```bash
# Запустить тесты
run-tests.cmd

# Запустить приложение
run-app.cmd
```

**Linux/Mac:**
```bash
# Запустить тесты
./run-tests.sh

# Запустить приложение
./run-app.sh
```

## 🔧 Полезные команды

### Установка зависимостей
```bash
mvn clean install
```

### Запуск тестов
```bash
mvn clean test
```

### Запуск приложения
```bash
mvn spring-boot:run
```

### Сборка без тестов
```bash
mvn clean package -DskipTests
```

### Проверка зависимостей
```bash
mvn dependency:tree
```
