## 🎨 Функциональность

### Авторизация ✅
- Регистрация пользователей с валидацией
- Вход с email и пароль
- BCrypt шифрование паролей
- Безопасность доступа (Spring Security)

### Проекты ✅
- Создание проектов с названием, описанием и датами
- Просмотр списка своих проектов
- Редактирование проектов
- Удаление проектов
- Изоляция данных - пользователь видит только свои проекты

### Задачи ✅
- Создание задач в проектах
- 4 статуса: Запланирована, В работе, Выполнена, Заблокирована
- Прогресс от 0 до 100%
- Отслеживание исполнителя
- Редактирование и удаление задач
- Автоматический прогресс при изменении статуса на DONE

### Диаграмма Ганта ✅
- Визуализация задач как горизонтальных полос
- Цветовая кодировка по статусам
- Отображение прогресса выполнения
- Адаптивная временная шкала
- Собственная реализация (без внешних библиотек)

### Дизайн ✅
- Современный и чистый интерфейс
- Адаптивный дизайн (работает на мобилах, планшетах, ПК)
- Все тексты на русском языке
- Пустые состояния и сообщения об ошибках
- Профессиональная цветовая схема

### Безопасность ✅
- Пароли хешируются через BCrypt
- CSRF защита
- Защита доступа - невозможно открыть чужой проект
- SQL Injection защита (параметризованные запросы)
- Проверка доступа на уровне сервиса и контроллера

## 🚀 Как запустить

### Быстрый способ (рекомендуется)

**Windows:**
```bash
cd C:\Users\diana\OneDrive\Рабочий стол\GanttPRO
run-app.cmd
```

**Linux/Mac:**
```bash
cd /path/to/GanttPRO
./run-app.sh
```

### Через Maven

**Windows:**
```bash
mvnw.cmd spring-boot:run
```

**Linux/Mac:**
```bash
./mvnw spring-boot:run
```

### После запуска

1. Откройте http://localhost:8080
2. Введите учетные данные:
   - Email: `demo@example.com`
   - Пароль: `demo123`
3. Начните работать с проектами и задачами!
